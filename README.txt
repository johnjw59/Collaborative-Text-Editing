Instructions:

1. Run front-end.go with the specified arguments
2. Run replica.go in storage mode by inputting the replica id argument to be "storage"
3. Run replica.go in client mode by inputting the replica id argument to be any other string other than "storage"
4. Navigate to localhost:8080 in your browser to view the web app

Notes:
- To retrieve a specific document, specify the URL path: localhost:8080/doc/<document-id>
